<h3>Run Locally</h3>

<p>Clone the project</p>

```bash
  git clone https://github.com/RuanDevJs/vertigo-sign-in
```

<p>Go to the project directory</p>

```bash
  cd vertigo-sign-in
```

<code>Install dependencies</code>

```bash
  yarn or npm install
```

<code>Start the server</code>

```bash
  yarn start or npm run start
```
